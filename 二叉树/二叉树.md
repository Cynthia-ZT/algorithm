二叉树与递归
1. 如何思考二叉树相关问题
# 不要一开始就陷入细节
# 而是思考整棵树与其左右子树的关系
# 原问题：计算整棵树的最大深度
# 子问题：计算左/右子树的最大深度

2. 为什么需要使用递归
# 子问题与原问题是相似的
# 类比循环，执行的代码也应该是相同的
# 但子问题需要把计算结果返给上一级问题
# 这更适合用递归实现
# 由于子问题的规模比原问题小，不断递下去，总会有尽头
# 即递归的边界条件(base case)，直接返回它的答案(归)

3. 为什么这样写就一定能算出正确答案
# 数学归纳法
# 类似多米诺骨牌，一个接一个倒下的过程

4. 计算机是怎么执行递归的

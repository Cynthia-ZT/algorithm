"双指针"顾名思义，就是用两个变量（指针）同时在一个序列上移动。它能让我们在 O(n) 的时间内解决一些原本要 O(n²) 的问题。

🧭 一、同向双指针（又叫“快慢指针”）

特点：两个指针都从同一方向出发，通常都往同一个方向移动。
常见用途：
    删除重复元素
    滑动窗口（找子数组/子串）
    链表中判断是否有环（快慢指针）
例子：去重
```
nums = [1, 1, 2, 2, 3]
slow = 0
for fast in range(1, len(nums)):
    if nums[fast] != nums[slow]:
        slow += 1
        nums[slow] = nums[fast]
# 结果：前 slow+1 个就是去重后的部分
```
👉 fast 负责“探索”新元素，slow 负责“保留”不重复的结果。
两指针方向相同，速度不同。

⚖️ 二、双向双指针（又叫“对撞指针”）

特点：两个指针从序列的两端出发，向中间靠拢。
常见用途：
    判断是否为回文串
    有序数组找两数之和
    排序数组中的区间查找

例子（两数之和）：
```
nums = [1, 2, 4, 6, 10]
target = 8
left, right = 0, len(nums) - 1

while left < right:
    s = nums[left] + nums[right]
    if s == target:
        print(left, right)
        break
    elif s < target:
        left += 1
    else:
        right -= 1
```
👉 两指针方向相反，一个从左走，一个从右走。
目标是让它们在中间“碰头”或相遇。

💡 实用判断法：

如果问题涉及“相邻元素关系”或“顺序处理” → 想同向双指针
如果问题涉及“首尾对称”或“从两边收敛” → 想双向双指针
